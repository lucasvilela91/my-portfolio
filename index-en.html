About me

Hello! My name is Lucas Vilela, I am a Computer Engineering undergraduate student and have been passionate about technology since childhood. My journey began with a Super Nintendo, where I spent hours exploring worlds in the classic Super Mario World. When I got my first computer, I shared usage time with my brother. Together, we learned to explore every feature, and each discovery was an incredible and memorable experience.

I have experience in HTML, CSS, and JavaScript, and I am deepening my knowledge in libraries like React. Additionally, I am expanding my skills in mobile development with Swift, aiming to create applications that provide amazing experiences across different platforms.

Currently, I am focused on gaining more practical experience through projects involving the creation of web and mobile applications. In each project, I apply concepts of usability, good programming practices, and collaboration.

I enjoy working in a team and believe that idea sharing and experiences are essential for the success of any project.
I am curious, dedicated, and believe that technology is a powerful tool to transform the world.

Shall we build the future together?

Skills

html5

HTML, or HyperText Markup Language, is a markup language that allows you to create the basic structure of a web page.

css3

CSS (Cascading Style Sheets) is a stylesheet language that allows you to format the appearance of web pages written in HTML or XML.

javascript

JavaScript allows you to create interactive web pages, applications and online systems. It is one of the core technologies of the World Wide Web.

typescript

TypeScript is a superset of JavaScript that adds optional static typing capabilities, making it easier to create more robust code.

node.js

Node.js is a JavaScript runtime environment that allows you to execute JavaScript code on the server side. Node.js is built on Google Chrome's V8 engine.

react

React is an open-source JavaScript library for creating user interfaces (UI) in web applications, developed by a team at Facebook.

tailwind css

It is a CSS styling framework that provides predefined classes for styling web pages.

swift

Swift is a powerful and intuitive programming language from Apple for iOS, macOS, watchOS, and tvOS.

git

Git is a free and open source tool for tracking changes to source code during software development. It is a version control system.

github

It is a collaborative development platform that allows you to store, share and work on projects.

mysql

MySQL is a free and open source relational database management system that uses the SQL language.

MongoDB

MongoDB is an open source, non-relational document database that offers flexibility, scalability, and performance for modern applications.

npm

Npm (Node Package Manager) is the default package manager for Node.js. It is used to install, manage, and share JavaScript libraries and tools.

sass

Sass (Syntactically Awesome Stylesheets) is a CSS preprocessor that adds features like variables, nesting, mixins, and inheritance, making code more organized and reusable.

java

Java is an object-oriented language created in 1995 by Sun Microsystems, used for web, desktop and mobile. It currently belongs to Oracle and is widely used in corporate systems.
